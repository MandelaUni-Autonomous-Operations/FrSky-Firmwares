# FrSky Firmware
##  Transmitter Firmware
###  Q X7 (ACCST)
See the folder: [FW-XJT-v2.1.0](https://github.com/MandelaUni-Autonomous-Operations/FrSky-Firmwares/tree/main/FW-XJT-v2.1.0)
The FCC version of the firmware should be run on all X7 radios in the lab. To flash the transmitter's internal RF module, place the FCC.frk file in the firmware folder of the SD card.
On the radio, navigate to the firmware folder, click on the file and select "Flash Int Module".

## Radio Firmware
### RX8R Pro
See the folder: [FW-RX8R_Pro-v2.1.0](https://github.com/MandelaUni-Autonomous-Operations/FrSky-Firmwares/tree/main/FW-RX8R_Pro-v2.1.0)
The FCC version of this firmware should be flashed to the radio. To flash using an X7, use a normal servo lead to connect the smart port of the radio to the smart port of the transmitter, as shown below.
